注: 此文档分大类描述不同接口的返回码。具体返回码代表的含义将作以下格式描述.

  返回码数字 -- 调用的接口 -- 解释

1. 全局错误码
  0 -- 成功
  -1 -- 请求参数不正确

2. 物品类别接口
  1101 -- /thing/detail?id=xx -- 不存在此物品类别

3. 用户信息处理接口
  1201 -- /user/login -- 更新数据库失败。
  1202 -- /user/info -- 用户不存在(在更新用户联系方式的时候)
4. 获取临时Token接口
    1301 -- /tencent/coskey -- 获得腾讯服务器端响应失败。
    1302 -- /tencent/coskey -- 无法正确解析腾讯服务器返回的结果。
5. 查询指定学校的地标建筑物信息
    1401 -- /school/list 指定的学校ID不存在 
6. 添加/更新失物记录
    1501 -- /thing/update -- 不能更新丢失物的信息。无法从数据库中找到旧对象。这通常是由于提交待更新的对象中不存在GUID Field或者 GUID Field中的记录与数据库中的任何一条都不匹配。
    1502 -- /thing/update -- 不能更新丢失物的信息，原因是标记为不应该更改的条目遭到了更改。
    1503 -- /thing/update && /thing/publish -- 不能更新/创建丢失物的信息，原因是向数据库中写入记录失败。
    1504 -- /thing/update -- 不能更新丢失物的信息，原因是没有正确传入需要更新的对象
    1505 -- /thing/publish -- 不能发布丢失物信息，原因是没有正确传入需要发布的对象
